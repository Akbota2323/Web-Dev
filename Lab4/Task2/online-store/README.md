# Online Store - Lab 4

## Как запустить проект:
1. Склонируйте репозиторий.
2. В терминале выполните команду: `npm install`
3. Запустите сервер: `ng serve`
4. Откройте браузер по адресу: `http://localhost:4200/`

## Описание
Данное приложение является онлайн-витриной товаров с Kaspi.kz, разработанным в рамках Лабораторной работы №4.
